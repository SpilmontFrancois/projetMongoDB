### Installation & Lancement

- `npm install`

- `docker-compose up`

- `npm start`

- Ouvrir index.html dans un navigateur

- Cliquer sur le bouton "Actualiser les données"

Si une erreur survient sur le serveur, c'est que le docker n'est pas lancé complètement.
